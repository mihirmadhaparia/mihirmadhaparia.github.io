INTERACTIVE WEB APPStructural concept generation

AI Steel Structure Studio

Streamlit-based conceptual steel building generator

A browser-based concept generator that turns plain-English building prompts into a configurable steel structure package with 3D preview, conceptual drawings, BOM exports, and editable section catalogs.

  • Generates a 3D browser preview with primary steel, rafters, purlins, girts, cladding surfaces, bracing, connection elements, and slab geometry.
  • Exports STL, OBJ, DXF, SVG, BOM CSV, and JSON packages for review, downstream development, and client-facing concept work.
  • Supports free built-in prompt parsing plus optional Ollama, OpenAI, and Gemini extraction backends.
  • Uses editable steel, purlin, brace, and roofing catalogs so it can grow into a deeper CAD and estimating workflow.
